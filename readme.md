## 2. ShortCuts

### 2.1. web/h5/微信公众号

功能|快捷键
---|---
新增页面| `alt+p`
新增组件| `alt+n`
新增服务| `alt+s`
新增响应| `alt+a`
新增渲染块| `alt+x`
添加原子操作|`alt+t a`
添加路由|`alt+r`
添加控件|`alt+t w`
添加通用模板|`alt+t t`
启动调试|`alt+d`
项目打包|`alt+b`
创建项目|`alt+c`
