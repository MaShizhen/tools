## 3. 项目类型支持

|类型|支持情况|
|---|---|
|`next`|支持|
|`taro`|支持|
|`uniapp`|尽可能支持|
|`desktop`|虽然保留，但暂无精力维护|
|`web/h5`|废弃，转`next`|
|`mobile`|废弃，转`taro`|
|`wx`|废弃，转`taro`|

## 4. 支持的操作

操作名称|默认快捷键
---|---
新增项目|`alt+m p`
新增页面|`alt+p`
新增原子操作|`alt+a`
插入原子操作|`alt+t a`
新增通用原子操作|`alt+m alt+a`
开启开发模式|`alt+m d`
打包|`alt+m b`
查找|`alt+m f`
