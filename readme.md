## 3. 项目类型支持

|类型|支持情况|
|---|---|
|`next`|支持|
|`taro`|支持|
|`uniapp`|尽可能支持|
|`desktop`|虽然保留，但暂无精力维护|
|`web/h5`|废弃，转`next`|
|`mobile`|废弃，转`taro`|
|`wx`|废弃，转`taro`|
